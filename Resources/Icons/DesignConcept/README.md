# アイコンデザインコンセプト

## 技術仕様

### アイコンファイル規格
- **解像度**: 96×96 ピクセル
- **DPI メタデータ**: 288 DPI（PNG の pHYs チャンク）
- **WPF 論理サイズ**: 96 ÷ (288 ÷ 96) = **32 論理ピクセル**（リボンボタン枠にちょうど収まる）
- **ファイル名規則**: `{name}_96.png`
- **フォーマット**: PNG（透過背景）

### 描画スペースの考え方
- 32×32 の論理座標系で設計し、`s(v) = v * 3` で 96px にスケール
- 座標・サイズはすべて `s()` 関数を通す（例: `s(16)` = 中央）

### 生成ツール
- **メインツール**: `Tools/IconGenerator96.html`（ブラウザで開いてプレビュー＆ダウンロード）
- **DPI 設定**: ツール内で pHYs チャンクを自動付与（288 DPI）
- **Python 生成**: `Tools/generate_icons.py`（`python3 Tools/generate_icons.py` で全 PNG 一括出力）

## シート表現の設計ルール（sheet_creation / viewport 系）

### 縦横比
- **A版用紙の縦横比（√2:1）に準拠**
- 32 座標系での標準サイズ: **幅 26 × 高さ 18**（= 1.44:1 ≈ √2）

### 線の太さ（統一）
- シート外枠線・図枠線・表題欄線： すべて **`s(0.6)`** で統一
- コピー/ペーストの区別用ビューポート線のみ **`s(1.2)`**

### コピー vs ペーストの区別
- **コピー**: 破線（`setLineDash([s(2.5), s(1.5)])`）
- **ペースト**: 実線
- 両方とも線幅は同じ `s(1.2)`

## 各アイコンのデザイン概要

### 通り芯・レベル パネル

#### `both`（両端表示）
- 2本の縦線 + 2本の横線（格子状）
- 各交点にバブル（白丸、黒縁 R=3）
- 線はバブルの縁から縁まで（はみ出さない）

#### `left`（左端のみ）
- 上2つ + 左2つのバブル（合計4個）
- 線の右端・下端はバブルなしで canvas 端まで伸ばす

#### `right`（右端のみ）
- 下2つ + 右2つのバブル（合計4個）
- 線の左端・上端はバブルなしで canvas 端まで伸ばす

---

### シート パネル

#### `sheet_creation`（シート一括作成）
- **3枚のシートが斜め方向に重なる**表現
- 奥から手前へ: LIGHT → MID → DARK の線色で奥行き感
- 最前面シートのみ図枠線 + 表題欄を描画
- シート配置（各 26×18）: 3枚目(6,5)、2枚目(3,8)、1枚目(0.5,11)

---

### ビュー パネル

#### `viewport_copy`（ビューポート位置コピー）
- シート1枚（A版比率 31×22）+ 図枠線 + 表題欄
- ビューポートを表す **青色破線の四角**（シート左寄り 4,9〜18,19）
- 破線: `[s(2.5), s(1.5)]`

#### `viewport_paste`（ビューポート位置ペースト）
- シート1枚（A版比率）+ 図枠線 + 表題欄
- ビューポートを表す **青色実線の四角**（位置・サイズはコピーと同じ）

#### `view_copy`（3Dビューコピー）
- アイソメトリック立方体（淡い青系3色）、6頂点定義（Vt/Vtr/Vtl/Vc/Vbr/Vbl/Vb）
- cx=16, cy=15, dx=12, dy=7
- 面: 上面 `#E0EEF8`、左面 `#B0CCE0`、右面 `#88B0CC`、辺: `#2A4A6A`

#### `view_paste`（3Dビューペースト）
- アイソメトリック立方体（濃い青系3色）、同じ6頂点定義
- 面: 上面 `#5FA8D8`、左面 `#2A7AB8`、右面 `#145890`、辺: `#0A2040`

#### `sectionbox_copy`（セクションボックスコピー）
- **キャビネット投影**の青破線直方体ワイヤーフレーム（12辺）+ 内部小直方体（グレー淡色3面）
- 外枠: 前面 `(2,8)-(25,30)` = 23×22、奥行き `(+5,-6)` → 後面 `(7,2)-(30,24)`
- 描画順: ①後面4辺+奥行き線4辺（破線） → ②内部ソリッド → ③**前面4辺（最前面）**
- 破線: `[2.5, 1.5]`、線幅 `s(1.5)`、色 `#0066CC`
- 内部小直方体: 前面 `(9,14)-(18,23)` = 9×9、奥行き `(+4,-4)`
- 内部色: 上面 `#A8B8C8`・右面 `#7A8A9A`・前面 `#B8C8D8`

#### `sectionbox_paste`（セクションボックスペースト）
- **キャビネット投影**の青実線直方体ワイヤーフレーム（12辺）+ 内部小直方体（青濃色3面）
- 外枠・奥行きパラメータは copy と同じ、実線、線幅 `s(1.5)`、色 `#2288DD`
- 描画順: copy と同じ
- 内部色: 上面 `#5090D0`・右面 `#2870B0`・前面 `#70B0E0`

#### `cropbox_copy`（トリミング領域コピー）
- **`both`（通り芯両端）パターンをベース** に青破線矩形を重ねる
- バブル半径 R=2、矩形 (8,8)-(24,24)
- 描画順: ①グリッド線(DARK) → ②バブル → ③青破線矩形
- 破線: `[2.5, 1.5]`、線幅 `s(1.2)`

#### `cropbox_paste`（トリミング領域ペースト）
- **`both`パターンをベース** に青実線矩形を重ねる
- バブル半径 R=2、矩形 (8,8)-(24,24)
- 実線、線幅 `s(1.2)`

### 注釈・詳細 パネル

#### `room_tag`（部屋タグ一括配置）
- 表形式アイコン（罫線グリッド、外枠 1.5〜30.5）
- 上部1行は横結合（通し行、y=1.5〜11）、下部3列グリッド（列境界 x=9.5, x=20）
- 最下行（y=26.5〜30.5）は縦罫線なし（1セル）

#### `filled_region`（塗潰し領域 分割・統合）
- 左矩形: クロスハッチ（＼＋／）、右矩形: 単方向斜め線（＼）
- 下部に「← →」の矢印（分割・統合を表現）、step=4、線幅 s(0.7〜0.8)

### 構造 パネル

#### `beam_under_level`（梁下端色分け）
- I型梁（上端y=1、下端y=17）+ 上向き矢印（先端=梁下端y=17）+ 一点破線FL（y=27）+ ▼マーカー + 右側3色ブロック
- 3色ブロック: x=22〜30、y=1/11/21、各8×8単位、ピンク/黄/青

#### `beam_top_level`（梁天端色分け）
- 一点破線FL（y=5）+ ▼マーカー + 下向き矢印（先端=梁上端y=15）+ I型梁（y=15〜31）+ 右側3色ブロック
- 3色ブロック: x=22〜30、y=1/11/21、各8×8単位、ピンク/黄/青

#### `fire_protection`（耐火被覆色分け）
- 外周矩形（被覆・ライトグレー `#C8C8C8`）内にI型梁断面（ダークグレー `#505050`）
- 外枠: (1,1)-(19,31)、上フランジ (2.5,4)-(17.5,8.5)、ウェブ (7.5,8.5)-(12.5,23.5)、下フランジ (2.5,23.5)-(17.5,28)
- 右側3色ブロック: x=21〜30、y=2/12/22

#### `formwork`（型枠数量算出）

**3D型枠 + 集計表 + Σ の構成**

```
座標パラメータ（32単位系）
  奥行きベクトル: ddx=20, ddy=-4  ← 右方向に強調
  コンクリート: cx0=0, cy0=5, cx1=5, ybot=31
  コンパネ右端: cx1+ddx = x=25
```

描画レイヤー（前から後ろへ）:

| レイヤー | 内容 | 色 |
|---------|------|-----|
| ① コンクリート側面 | 矩形 x=0〜5, y=5〜31 | `rgb(175,175,170)` |
| ② コンクリート天端 | 台形 (0,5)-(5,5)-(25,1)-(20,1) | `rgb(215,215,210)` |
| ③ コンパネ面 | 平行四辺形 x=5〜25 | `rgb(240,195,155)` オレンジ |
| ④ 縦バタ桟木×3 | t=0/0.5/1.0 各tw=0.22幅、3色立体 | 明 `rgb(210,158,75)` / 面 `rgb(158,100,38)` / 影 `rgb(105,62,18)` |
| ⑤ 横端太×3 | fy=0.25/0.50/0.75、各wh=1.5高さ、3色 | 明 `rgb(215,220,228)` / 面 `rgb(162,167,176)` / 影 `rgb(95,100,112)` |
| ⑥ セパレータ | コンクリート正面の破線（x=0.4〜4.8） | `rgb(100,100,98)` |
| ⑦ フォームタイ | 各端太位置に0.9×1.0の矩形 | `rgb(55,55,58)` |
| ⑧ Σボックス | 表と同サイズ（9.5×7.5単位）の白地＋白外縁取り | 白背景、s(10.5)フォント、`rgb(20,45,115)` |
| ⑨ 集計表 | ヘッダー(青灰)+データ(白)+合計(緑)の3行均等 | 行高 ROW_H=2.5 |

**集計表の詳細**:
```
TBX=22, TBW=9.5, TBY=22
行高: ROW_H=2.5（全行均等）
列境界: COL2=TBX+5.0=27
外縁取り: 白 s(1.4)（外側のみ）、内枠: ダーク s(0.35)（全罫線統一）
```

**ΣボックスとΣの詳細**:
```
位置: TBX=22〜31.5（表と同じX範囲）
高さ: TBY-gap〜TBY（表と同じ7.5単位）
SX=26.75（表中央）, SY=14.75
フォント: Georgia Bold s(10.5)=31.5px
色: rgb(20,45,115)（濃紺）
外縁取り: 表と同じ白 s(1.4)
```
