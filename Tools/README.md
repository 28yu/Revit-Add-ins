# Tools - アイコン生成ツール

## IconGenerator.html

### 概要
Tools28 Revitアドイン「塗潰し領域 分割/統合」機能のリボンアイコンを生成するHTMLツールです。

### アイコン仕様
- **デザイン**: 4つの正方形（ハッチング付き）を2×2配置
- **サイズ**: 32×32px
- **正方形サイズ**: 6×6px
- **枠線**: 0.6px
- **ハッチング線**: 0.4px、間隔1.8px
- **レンダリング**: 64×64pxで描画 → 32×32pxに高品質ダウンスケール
- **色**: 黒（#000000）

### 使用方法（Windows環境で実行）

#### 1. アイコン生成

```powershell
# ブラウザでHTMLファイルを開く
start Tools\IconGenerator.html
```

またはエクスプローラーで `Tools\IconGenerator.html` をダブルクリック

#### 2. ダウンロード

ブラウザで開いたページの「32×32をダウンロード」ボタンをクリック
→ `filled_region_32.png` がダウンロードフォルダに保存されます

#### 3. ファイル配置

ダウンロードした `filled_region_32.png` を以下の場所にコピー（上書き）:

```
Resources\Icons\filled_region_32.png
```

#### 4. ビルド＆デプロイ

```powershell
# クイックビルド（Revit 2022のみ）
.\QuickBuild.ps1

# または全バージョンビルド
.\BuildAll.ps1
```

#### 5. 動作確認

```powershell
# Revit 2022を起動
# → "28Tools" タブ → "詳細" パネル → "領域" ボタンのアイコンを確認
```

Revit起動中の場合は、一度終了して再起動してください。

### トラブルシューティング

#### アイコンが表示されない
- Revitを完全に終了して再起動
- ファイルパスが正しいか確認: `Resources\Icons\filled_region_32.png`
- PNGファイルのサイズが32×32pxであることを確認

#### ダウンロードできない
- ブラウザのポップアップブロックを無効化
- 別のブラウザ（Chrome、Edge、Firefox）で試す

### 技術詳細

#### 2xレンダリング
Canvas API で64×64pxで描画し、32×32pxにダウンスケールすることで、
アンチエイリアシングによる高品質なアイコンを生成します。

#### ハッチングパターン
45度の斜線を0.4px幅、1.8px間隔で描画し、
塗潰し領域（Filled Region）の典型的な表現を実現しています。
