# 言語切替・設定 使い方ガイド

28 Tools のUI言語を日本語・英語・中国語（簡体字）の3言語に切り替える機能です。リボンのボタン名・ツールチップ・全ダイアログのテキストが即座に切り替わります。また、バージョン情報の確認とマニュアルへのアクセスも同じパネルから行えます。

---

## できること

- 28 Tools のUI言語を**日本語 / 英語 / 中国語（簡体字）**の3言語から切り替えます
- 言語設定は**Revit再起動後も保持**されます（設定ファイルに保存）
- リボン上のボタン名・ツールチップ・全WPFダイアログのテキストが**即座に更新**されます
- バージョン情報（バージョン番号・有効期限）を確認できます
- 使い方マニュアルをブラウザで開けます

---

## 設定パネルの場所

リボン「**28 Tools**」タブの**右端パネル（設定）**

---

## 各コントロールの説明

### 言語切替（プルダウンボタン）

| 選択肢 | 言語 |
|---|---|
| **JP** 🇯🇵 | 日本語（デフォルト） |
| **US** 🇺🇸 | English |
| **CN** 🇨🇳 | 简体中文 |

プルダウンをクリックして言語を選ぶと、即座にUIが切り替わります。

> 💡 言語設定は DLL と同じフォルダの `28tools_lang.txt` に保存されます。Revitを再起動しても選択した言語が維持されます。

### バージョン情報（ⓘ ボタン）

- 28 Tools の**バージョン番号**と**有効期限**が表示されます
- 有効期限の30日前から、起動時に更新を促す通知が表示されます

### マニュアル（? ボタン）

- ブラウザが開き、使い方マニュアルページ（https://28tools.com/addins.html）にアクセスします

---

## 多言語化の対象範囲

以下のすべてのUI要素が3言語に対応しています。

| 対象 | 内容 |
|---|---|
| **リボンパネル名** | 通り芯・レベル、シート、ビュー、注釈・詳細、構造、データ、設定 |
| **リボンボタン名** | すべてのボタンのテキスト |
| **ツールチップ** | すべてのボタンのツールチップ |
| **WPFダイアログ** | タイトル・ラベル・ボタン・グループヘッダー（全8種のダイアログ） |
| **TaskDialogメッセージ** | エラー・確認・完了などのポップアップメッセージ |

---

## トラブルシューティング

### 言語を切り替えても一部のテキストが変わらない

- WPFダイアログのテキストはダイアログを**次回開いたとき**に反映されます
- 現在開いているダイアログは、いったん閉じてから再度開いてください

### バージョン情報ダイアログに「有効期限切れ」と表示される

- 現在インストールされている 28 Tools が有効期限を過ぎています
- 最新バージョンを https://28tools.com/addins.html からダウンロードしてインストールしてください

### マニュアルが開かない

- 既定のブラウザが設定されているか確認してください
- インターネット接続のないオフライン環境では開けません
