# 塗潰しパターン入出力

開いている Revit ファイル内の塗り潰しパターン（フィルパターン）を一覧表示し、
`.pat` ファイルへの書き出し（エクスポート）と、外部 `.pat` ファイルからの読み込み
（インポート）を行う機能です。

パネル: **注釈・詳細** / ボタン: **塗潰しパターン**

## 主な用途

- あるプロジェクトで作成した塗り潰しパターンを、別のプロジェクトへ持ち運ぶ
- AutoCAD 等で使われる `.pat` パターンを Revit に取り込む
- 社内標準のパターンをファイルとして共有・管理する

## 使い方

ボタンをクリックするとダイアログが開きます。

1. **一覧表示**
   モデル内の塗り潰しパターンが一覧表示されます。上部の **表示** 切り替えで
   「すべて / モデル / 製図」を絞り込めます。各行に名前・種類（モデル／製図）・
   グリッド数が表示されます。

2. **単位の選択**
   `.pat` ファイルの単位系を **mm / inch** から選択します（既定は mm）。
   エクスポート時はこの単位でファイルに書き出し、インポート時はファイルに
   `;%UNITS=` ヘッダーが無い場合の既定単位として使われます。

3. **エクスポート（書き出し）**
   一覧のチェックボックスで書き出したいパターンを選択し、**エクスポート** を
   クリックします。
   - 1 つだけ選択した場合: 保存ファイル名を指定して書き出します。
   - 複数選択した場合: 保存先フォルダを選び、各パターンを
     `パターン名.pat` として個別のファイルに書き出します。
   - ソリッド（べた塗り）パターンはグリッドを持たず `.pat` で表現できないため、
     書き出し対象から除外されます。

4. **インポート（読み込み）**
   **インポート** をクリックし、`.pat` ファイル（複数選択可）を選びます。
   ファイル内のパターン定義がモデルに追加されます。
   - 同名・同種のパターンが既に存在する場合はスキップします。
   - グリッドが無い等、作成できないパターンはスキップします。
   - 読み込み後、一覧は自動で更新されます。

## `.pat` フォーマットについて

Revit / AutoCAD 準拠のハッチパターン形式です。

```
;%UNITS=MM
*パターン名,Tools28
;%TYPE=DRAFTING
角度, X原点, Y原点, シフト, オフセット[, ダッシュ1, ダッシュ2, ...]
```

- `;%UNITS=MM` または `;%UNITS=INCH` … 単位系
- `;%TYPE=DRAFTING` または `;%TYPE=MODEL` … パターン種類
- グリッド行 … 角度・原点・シフト・オフセットと、任意のダッシュ（破線）定義

## 注意事項

- ソリッド塗り潰し（べた塗り）は `.pat` 形式では書き出せません。
- 単位ヘッダーの無い純粋な AutoCAD `.pat` を取り込む場合、ダイアログで選択した
  単位で読み込まれます。意図した寸法にならない場合は単位を切り替えて再読込
  してください。
