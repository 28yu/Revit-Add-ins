# 通り芯・レベル 符号表示切替 使い方ガイド

ビュー内のすべての通り芯（グリッド）とレベルの符号（バブル）表示を、ワンクリックで一括切り替えする機能です。

---

## できること

- ビュー内のすべての**通り芯（グリッド）**と**レベル**の符号表示を一括で切り替えます
- 3つのモードから選択できます：

| ボタン | 動作 |
|---|---|
| **両端表示** | 通り芯・レベルの両端に符号を表示する |
| **左/上のみ** | 左端（縦通り芯）または上端（横通り芯・レベル）のみ符号を表示する |
| **右/下のみ** | 右端（縦通り芯）または下端（横通り芯・レベル）のみ符号を表示する |

## 実行手順

### ▶ ステップ1: 対象ビューを開く

符号表示を変更したいビューをアクティブにします。

### ▶ ステップ2: ボタンをクリック

リボン「**28 Tools**」タブ →「**通り芯・レベル**」パネルの3つのボタンから目的のものをクリックします。

- **両端表示** — 通り芯・レベルの両端すべてに符号を表示
- **左/上のみ** — 片側（左端・上端）のみ表示
- **右/下のみ** — 片側（右端・下端）のみ表示

ダイアログは表示されません。クリックすると即座にビュー内のすべての通り芯・レベルに反映されます。

> 💡 選択不要です。ビュー内のすべての通り芯・レベルがまとめて切り替わります。

---

## トラブルシューティング

### クリックしても変化がない

- ビュー内に通り芯・レベルが**表示されているか**確認してください（フィルタやビュー設定で非表示になっていませんか？）
- すでに指定したモードになっている場合、見た目は変わりません

### リンクモデルの通り芯・レベルが変わらない

- リンクファイル内の通り芯・レベルは編集対象外です。ホストモデルの通り芯・レベルのみ変更されます

### 一部の通り芯だけ変更したい

- このコマンドはビュー内の全要素を一括で変更します。個別に変更したい場合は、Revit標準の通り芯編集操作を使用してください
