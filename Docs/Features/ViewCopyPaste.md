# ビュー コピー＆ペースト機能 使い方ガイド

ビューの設定（視点・切断ボックス・ビューポート位置・トリミング領域）をコピーして別のビューやシートに貼り付ける機能群です。いずれも**コピー → ペースト**の2ステップで操作します。

---

## 機能一覧

| 機能 | ボタン名 | 対象ビュー | コピーされる内容 |
|---|---|---|---|
| [3Dビュー視点](#3dビュー視点-コピーペースト) | 3Dビュー コピー / ペースト | 3Dビュー → 3Dビュー | カメラ位置・注視点・上方向 |
| [切断ボックス](#切断ボックス-コピーペースト) | SBコピー / SBペースト | 3Dビュー → 3Dビュー | セクションボックスの範囲 |
| [ビューポート位置](#ビューポート位置-コピーペースト) | VP位置コピー / VP位置ペースト | シート → シート | ビューポートの座標（ビュー名でマッチング） |
| [トリミング領域](#トリミング領域-コピーペースト) | トリミングコピー / トリミングペースト | 全ビュー → 全ビュー | クロップボックスの範囲・形状 |

リボン: 「**28 Tools**」タブ →「**ビュー**」パネル

> 💡 **共通事項**: コピーした情報はすべてセッション内のみ有効です。Revit を閉じると消えます。

---

## 3Dビュー視点 コピー＆ペースト

3Dビューのカメラ位置・向きを別の3Dビューにコピーします。視点を揃えたい複数の3Dビューがある場合に便利です。

### 実行手順

**① コピー**: 元の3Dビューをアクティブにして「**3Dビュー コピー**」をクリック

**② ペースト**: 適用先の3Dビューをアクティブにして「**3Dビュー ペースト**」をクリック

### 注意事項

- 3Dビュー同士でのみ使用できます（平面図・断面図は不可）
- セクションボックスの範囲は含まれません。範囲コピーは「**切断ボックス**」機能を使用してください

---

## 切断ボックス コピー＆ペースト

3Dビューのセクションボックス（切断ボックス）の範囲を別の3Dビューにコピーします。同じ切り取り範囲を複数の3Dビューで共有したいときに便利です。

### 実行手順

**① コピー**: セクションボックスが設定された3Dビューをアクティブにして「**SBコピー**」をクリック

**② ペースト**: 適用先の3Dビューをアクティブにして「**SBペースト**」をクリック

> 💡 ペースト先でセクションボックスが無効になっていても、自動的に有効化して範囲を適用します。

### 注意事項

- コピー元のビューでセクションボックスが**有効**になっている必要があります
- 3Dビュー同士でのみ使用できます

---

## ビューポート位置 コピー＆ペースト

シート上のビューポートの位置を、他のシートの同名ビューポートに自動的に合わせます。複数のシートで同じビュー（例: 各階の平面図）を同じ位置に揃えたいときに便利です。

### 実行手順

**① コピー**: 基準シートを開き「**VP位置コピー**」をクリック → シート上の基準ビューポートをクリック

**② ペースト**: 揃えたい別のシートを開き「**VP位置ペースト**」をクリック

ペースト時は、コピーしたビューポートの**ビュー名が一致するビューポート**が自動的に同じ位置に移動します。

> 💡 複数のシートに連続してペーストできます。同名ビューポートがあるシートを順番に開いてペーストするだけで全シートを揃えられます。

### 注意事項

- ビューポートの**サイズ・スケールは変更されません**。位置のみコピーされます
- ビュー名が完全一致するビューポートのみが移動対象です

---

## トリミング領域 コピー＆ペースト

ビューのトリミング領域（クロップボックス）の設定を別のビューにコピーします。同じ表示範囲を複数のビューで揃えるのに便利です。

### 実行手順

**① コピー**: トリミング領域が設定されたビューをアクティブにして「**トリミングコピー**」をクリック

**② ペースト**: 適用先のビューをアクティブにして「**トリミングペースト**」をクリック

> 💡 ペースト先でトリミングが無効になっていても、自動的に有効化して範囲を適用します。

### 注意事項

- **同じスケール・同じ参照レベル**のビュー間でコピーすると最も正確に一致します
- 3Dビューのセクションボックスのコピーは「**SBコピー / SBペースト**」を使用してください

---

## トラブルシューティング

| 症状 | 対処 |
|---|---|
| ペーストしても変化がない | 「コピー」を先に実行しているか確認。ペースト先のビュー種別が合っているか確認 |
| 3Dビューに適用できない | 3Dビュー視点・切断ボックスは3Dビュー以外には適用不可 |
| ビューポート位置が動かない | コピー元ビューのビュー名とペースト先シートのビューポートのビュー名が一致しているか確認 |
| トリミング後の表示範囲がずれる | コピー元とペースト先のスケール・参照レベルが同じか確認 |
| ペースト後にモデルが見えなくなった（切断ボックス） | セクションボックスの範囲外にモデルがあります。チェックをオフにして範囲を確認 |
| ビューテンプレートがあって変更できない | ビューテンプレートを一時解除してから実行してください |
