# アイコン画像の格納先

以下のファイル名で PNG 画像を配置してください。
`workflow-diagram.html` が自動で読み込みます。

| ファイル名 | 対象 | 推奨サイズ |
|---|---|---|
| `you.png` | あなた（開発者） | 112x112px |
| `claude.png` | Claude Code | 112x112px |
| `powershell.png` | PowerShell | 112x112px |
| `revit.png` | Revit | 112x112px |
| `github.png` | GitHub | 112x112px |
| `nuget.png` | NuGet | 112x112px |
| `actions.png` | GitHub Actions | 112x112px |
| `users.png` | ユーザー | 112x112px |

画像が未配置の場合は、絵文字のフォールバックアイコンが表示されます。
